FLOW LOGIKA PERHITUNGAN TIAP METODE:

SAW (Simple Additive Weighting):
  Input Matrix → Normalisasi → Pembobotan → Penjumlahan → Ranking

SMART:
  Input Matrix → Hitung Utility → Normalisasi Bobot → Penjumlahan → Ranking

Profile Matching:
  Input Nilai Aktual → Hitung GAP → Konversi ke Bobot → CF & SF → Nilai Aspek → Ranking

Goal Programming:
  Input Target & Kapasitas → Alokasi Berdasarkan Harga → Hitung Metrik → Evaluasi Target
